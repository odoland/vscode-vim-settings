# Default VSCode-VIM settings 
Installs:
- vscode-vim w/ configs
- vscode w/ custom keyboard bindings + extensions
- snippets for go and ts
- zsh + a few handy CLI 

# Prerequisites:
- [brew](https://brew.sh/)
- [vscode](https://code.visualstudio.com/)
- [iTerm2](https://iterm2.com/)

# Installation:
`./setup.sh`
